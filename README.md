# studious-octo-winner

A weekend experiment in keeping track of the neighborhood's stray cats.

It started when I noticed the same orange tabby showing up on my porch every
morning at 7am sharp, rain or shine. I named him Mr. Pancakes. Then came a
gray one I call The Accountant, because he sits very still and seems to be
judging my life choices.

This repo is where I log their visits, invent backstories for them, and keep
a running leaderboard of who's the most reliable breakfast guest. Current
standings: Mr. Pancakes in a commanding lead, The Accountant a distant second,
and a mysterious black cat known only as "??" who has appeared exactly once
and haunts my thoughts.

More features coming whenever the cats feel like cooperating.
