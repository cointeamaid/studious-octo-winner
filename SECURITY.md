# Security Policy

This is a hobby project — a handful of static HTML, CSS, and JavaScript files. There's no server, no database, and no accounts. Everything runs in your own browser, and the only thing it saves is your cat's stats in that browser's local storage. Nothing gets sent anywhere and there's no user data to leak.

## Reporting something

If you spot something that looks off — a way to break a page, a dependency worth flagging, anything at all — please open an issue. If you'd rather not do it in public, you can reach me through my GitHub profile instead.

I'll take a look when I get a chance. Thanks for looking out.
