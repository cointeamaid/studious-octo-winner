# Contributing

Thanks for even looking! This is mostly a personal playground, but if you want to poke at it, here's the lay of the land.

It's all plain HTML, CSS, and JavaScript. No framework, no build step, no npm install — clone it, open index.html, and you're running.

A few pointers:

- The Cat Companion logic and the little SVG cat are all in cat.js. The tunable numbers (how fast the stats drift, what each action does) live right at the top of that file.
- The cat roster — names, colors, personalities — is in cats.json. Add a cat there and it shows up everywhere automatically.
- The sighting tracker is in porch-cats.html + app.js.
- Please keep it dependency-free. Half the fun is that it's just files you can open anywhere.
- If you're planning something big, open an issue first so we don't both build the same thing.

That's really it. Be kind, and have fun with it.
